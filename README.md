# 🎯 RFM 模型优化研究：电商用户分层的误判识别与修正

> 基于 Kaggle 电商数据集，识别传统 RFM 模型的三大误判场景，提出融合收入与参与度的优化模型


## 📊 项目背景

传统 RFM 模型（Recency, Frequency, Monetary）是电商用户分层的经典方法，但在实际应用中存在明显的**价值误判**问题。本项目通过数据挖掘，识别出三类典型误判场景，并构建 **RFM-I 模型**（加入 Income 收入维度与 Engagement 参与度维度），显著提升用户分层的准确性。

**误判影响**：在 1000 个用户样本中，传统模型误判率高达 **20.5%**（205 人），可能导致高价值用户流失或营销资源错配。


## 📊 数据集

- **来源**：[Kaggle - Ecommerce Product Recommendation Collaborative](https://www.kaggle.com/datasets/kartikeybartwal/ecommerce-product-recommendation-collaborative?resource=download)
- **传统** RFM 模型：Hughes, A. M. (1994). *Strategic Database Marketing*
- **参与度指标设计**：参考 AIPL 模型与增长黑客理论-
- **数据规模**：1000 条用户记录，15 个字段
- **数据字段说明**：

| 字段 | 说明 |
|------|------|
| User_ID | 用户唯一标识 |
| Age | 年龄 |
| Gender | 性别 |
| Location | 地区（Urban / Suburban / Rural） |
| Income | 年收入 |
| Interests | 兴趣偏好 |
| Last_Login_Days_Ago | 距离最近一次登录的天数 |
| Purchase_Frequency | 购买频次 |
| Average_Order_Value | 平均订单价值 |
| Total_Spending | 累计消费金额 |

| Pages_Viewed | 平均浏览页数 |
| Newsletter_Subscription | 是否订阅简报 |


## 🔍 三大误判场景

### 1️⃣ 低消费土豪误判
| 特征 | 传统模型判定 | 实际价值 |
|:---|:---|:---|
| 高收入 + 低消费 | 一般价值/流失型 | 💎 高潜力待激活 |
| 消费力未释放，属于沉睡的优质资产 | | |

**误判逻辑**：传统模型只看消费金额，忽略收入背后的消费能力。

### 2️⃣ 高潜力浏览型误判
| 特征 | 传统模型判定 | 实际价值 |
|:---|:---|:---|
| 高浏览时长 + 高页面访问 + 低消费 | 一般发展/流失型 | 🚀 高转化潜力 |
| 活跃度高但转化低，属于待培育用户 | | |

**误判逻辑**：传统模型忽略行为参与度，将"逛不买"用户直接降级。

### 3️⃣ 忠诚用户价值低估（本项目无误判，仅为理论参考）
| 特征 | 传统模型判定 | 实际价值 |
|:---|:---|:---|
| 低收入 + 高消费占比 | 普通价值型 | ❤️ 高忠诚/高信任 |
| 消费占收入比极高，体现强平台依赖 | | |

**误判逻辑**：同金额消费对高低收入用户的意义不同，传统模型无法区分。


## 🛠️ 优化方案：RFM-I 模型

### 新增维度

| 维度 | 计算方式 | 权重 |
|:---|:---|:---|
| **I - Income** | 收入三分位数分层 | 辅助判断消费能力 |
| **E - Engagement** | `0.3×活跃度 + 0.25×浏览页数 + 0.25×停留时长 + 0.2×订阅状态` | 衡量参与深度 |

### 新用户分层体系

```
├─ 潜力浏览型（127人, 12.7%）→ 高活跃低转化，重点培育
├─ 土豪休眠型（65人, 6.5%）→ 高收入低活跃，激活消费
├─ 忠诚死忠型（11人, 1.1%）→ 低收入高贡献，维护忠诚
└─ [保留传统 RFM 分层] → 重要价值/发展/保持/挽留型等
```


## 📈 核心数据洞察

### 相关性发现（Pearson 系数）

| 发现 | 业务含义 |
|:---|:---|
| **收入 vs 消费呈弱负相关** | 高收入用户消费潜力未被挖掘 |
| **浏览行为 vs 消费几乎无关** | "只逛不买"是独立用户群体 |
| **购买频次 vs 客单价独立** | 可同时从"提频"和"提客单"发力 |

### 关键数据对比

| 用户类型 | 浏览时长(min) | 总收入 | 消费金额 | 策略重点 |
|:---|---:|---:|---:|:---|
| 潜力浏览型 | **505** | 中高 | 1,500 | 转化激活 |
| 土豪休眠型 | 180 | **高** | 1,600 | 高端推送 |
| 忠诚死忠型 | 320 | 低 | **4,000+** | 情感维护 |
| 传统重要价值型 | 250 | 中 | 3,500 | 常规保持 |

## 💡 业务应用策略

### 潜力浏览型（127人）—— 转化优先
- **即时策略**：首单优惠券、限时包邮、加购提醒
- **长期策略**：个性化推荐引擎、浏览历史分析

### 土豪休眠型（65人）—— 激活优先
- **即时策略**：高端商品推送、VIP 专属折扣、黑卡会员
- **长期策略**：尊享服务体系、高端社群运营

### 忠诚死忠型（11人）—— 维护优先
- **即时策略**：感谢信、周年礼、老用户专享价
- **长期策略**：推荐返利、UGC 激励、情感连接
